# Stroke Prediction Using KNN

## Overview
This project predicts stroke risk using the K-Nearest Neighbors (KNN) machine learning algorithm.

## Dataset
- Dataset: healthcare-dataset-stroke-data (Kaggle)
- Size: 5,111 patient records

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn

## Project Workflow
1. Data Preprocessing
2. Exploratory Data Analysis
3. Feature Selection
4. Model Training using KNN
5. Model Evaluation

## Results
- Accuracy: 76%
- Precision: 52.9%
- Recall: 36%
- F1 Score: 42.9%
